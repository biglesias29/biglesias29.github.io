Project Description

This project began as a simple ride-on toy car but was reimagined into a rugged support vehicle for a child with disabilities. The goal was to provide him with greater independence and mobility while maintaining safety and control. To accomplish this, we reprogrammed the system to accept both a physical foot pedal and input from an RC transmitter. Using a servo-based override, the caregiver can switch between manual and remote operation, offering flexibility based on the environment or situation. The entire system was upgraded with dual ESCs and reinforced electronics to handle rough terrain while still being responsive and accessible. This vehicle isn’t just fun—it serves a real purpose for families who need adaptive mobility options for their kids.

Tools & Components

  • 2 × 18V Power Tool Batteries (wired in parallel)
  • Hobbywing QuicRun 880 Dual-Motor ESC (rear motors)
  • Radiolink CL9030 ESC (steering motor)
  • RC Transmitter + Physical Pedal Control
  • Servo disconnect system for manual/RC control switch

Component Breakdown

Component Function
18V Drill Batteries (x2) Power supply in parallel for extended runtime
Hobbywing QuicRun 880 ESC Controls rear dual-motor drive system
Radiolink CL9030 ESC Controls steering motor independently
RC Transmitter + Receiver Provides wireless control with override option
Servo Disconnect Mechanism Switches between RC and manual pedal control

System Upgrade

We replaced the old onboard “computer” and its limited remote with a more reliable and responsive RC transmitter and receiver system. The new setup offers better range, smoother control, and easier pairing. Steering and throttle are now fully mapped to the new remote, and the vehicle toggles seamlessly between manual and RC control using a servo-based disconnect.

Status

The vehicle works reliably and is capable of supporting over 500 lbs. It features a RC toggle switch, hot swappable batteries, extended range.

Next Steps

The next step is to design a cleaner control interface by fully removing the old shifting hardware. I’m also planning to add a screen mount and night lights for visibility, along with a detachable hitch. I'm currently exploring more compact ESC options and a modular mounting solution for the electronics inside the body.
